# radiko Keyboard Player
ショートカットキーで radiko のプレイヤーを操作する Chrome 拡張

## キーバインド
| キー              | 動作                |
| :---------------- | :------------------ |
| `Enter`           | 再生/停止を切り替え |
| `R`               | 最初から再生        |
| `1` ~ `9`         | ×10秒スキップ       |
| `Alt` + `1` ~ `9` | ×10秒戻る           |
